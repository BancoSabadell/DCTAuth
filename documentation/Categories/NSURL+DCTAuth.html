dctAuth_URLByAddingQueryParameters:

Adds the parameters to the query component of the URL.

- (NSURL *)dctAuth_URLByAddingQueryParameters:(NSDictionary *)parameters

Parameters

parameters

The parameters to add.

Return Value

A modified NSURL with the query component altered.

Discussion

It will not remove existing query parameters, however if the same keys exist then the newer value will be used.
