Overview

DCTAuth is a library to handle multiple authentication types for services that use OAuth, OAuth 2.0 and basic authentication.

Tasks

setURLOpener:

Applications can supply their own webview UI for opening a web-based URL by providing a block to open the given URL and indictate by returning YES, that the URL has been shown to the user;

+ (void)setURLOpener:(BOOL ( ^ ) ( NSURL *URL ))opener

Parameters

opener

The block that will handle the display of the web page.

Discussion

The application should monitor when the browser loads pages and for each new request in the webview, should call handleURL: to see if that call is the callback that DCTAuth is waiting for.

Calling handleURL: will check to see if the URL is the callback so you can use the result of that to decide to show the new load or not.
