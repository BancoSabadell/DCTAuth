Instance Methods

signURLRequest:forAuthRequest:

A method to allow account subclasses to sign a URL request.

- (void)signURLRequest:(NSMutableURLRequest *)request forAuthRequest:(DCTAuthRequest *)authRequest

Parameters

request

The request to be signed.

authRequest

The DCTAuthRequest object that is asking for the request to be signed.

Discussion

DCTAuthAccount subclasses need to implment this method and use their credentials to modify the given mutable request so that it is authorized for the user’s account. For example in the case of OAuth 1.0, the account adds a signed Authorization header to the request, with the correct OAuth parameters.
